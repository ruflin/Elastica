Abstract helper class to implement search indices based on models.

This abstract model should help creating search index and a subtype with some easy config entries that are overloaded.

The following variables have to be set: - $_indexName - $_typeName

The following variables can be set for additional configuration - $_mapping: Value type mapping for the given type - $_indexParams: Parameters for the index
