Adds a file to the index

addFile(string $key, string $filepath, string $mimeType) : \Elastica_Document

To use this feature you have to call the following command in the elasticsearch directory:

./bin/plugin -install elasticsearch/elasticsearch-mapper-attachments/1.2.0

This installs the tika file analysis plugin. More infos about supported formats can be found here: http://tika.apache.org/0.7/formats.html
